# testinglib
Test Lib


Download
--------

Download [the latest JAR][2] or grab via Maven:
```xml
<dependency>
  <groupId>com.daver1419.testinglib</groupId>
  <artifactId>testinglib</artifactId>
  <version>0.0.2</version>
</dependency>
```
or Gradle:
```groovy
compile 'com.daver1419.testinglib:testinglib:0.0.2'
```
